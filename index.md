The recommended method of installing this library is using [Docker](https://docs.docker.com/get-started/get-docker/).

Run the following command from your project root:

```bash
$ docker run -ti --rm -v .:/app estratocloud/manifesting --help
```

Or you can manually download it from [github.com/estratocloud/manifesting](https://github.com/estratocloud/manifesting/releases)
